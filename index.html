Appilii

Akkam jirtuu dawaatoota keenya , guyyaa har'aa seenaa gooticha itoophiya tokko sinitti himna . gootichi kunis koloneel abdiisaa Aaga jedhama.

Appiliin fuduraa baay’ee beekamaa fi jaalatamaa yoo ta’u, qabbiyyeewwan kan akka viitaaminii, Anti oxidiyeentii garaa garaa akkasumas, wantoota goginsa garaa hanbisuuf fayyadanii fi nyaata madaalamaa garaa garaaf kan fayyadudha. Nuyutirentii garaa, garaa qabaachuu irra kan ka’e, dhukkuboota hedduu dhala namaa irraa ittisa. Appiliin Boca, halluu, dhamdhama, qabiyyee nuyutirentii fi faayidaa fayyummaaf oolan irratti hundaa’uun garaagarummaa qabu. Fkn kaanserii, Ulfaatina qaamaa garmalee, dhukkuba onnee, sukkaaraa fi kanneen biroo ittisuuf gahee guddaa qaba.

Faayidaalee appilii irraa argamu


Fayiverii fi Vitaaminii akkasumaas, Albuudotaan kan badhaadheedha innis Anti oxideyentii guddaa qabaachuu irrraa kan ka’e fayyaa dhala namaaf hedduu barbaadamaadha. Appiliin erga hamma kana faayidaa qabeessaa ta’ee biyya keenya Itiyoopiyaatti Oomishni isaas ta’ee nyaanni isaa baay’ee beekamaa miti. Sabaabni isaas appiliin gabaa biyyatiif dhihaatu Oomisha biyya Alaa waan ta’ef, gabaa irratti gatiin isaa Baay’ee Miyaawaadha Ykn Qaalawaadha. Kan irraa ka’uutiin namoonni hedduun akka fedha isaniitti bitanii itti fayyadamuun ulfaatadha. Hammaan xiqqaa ta’u ilee, biyya keenya kessatti Oomishamuu jalqabamee jira. Oomisha biyya keessaa kana nyaataaf oolichuu fi gabaaf dhiheessuuf dirqama lammii hundaati. Akkasumaas qulqullina fi qabiyyee isaa ol guddisuun, isa biyya alaatti oomishamu irraa gadi buusanii ilaaluu dhiisuun oomisha biyya keenyaa beeksisuu fi haalaan itti fayyadamuu gahee hunda keenyaati. Irra caalaas appilii biyya keenyatti oomishamu kan biyya hanbaa caalaa Organikiin kan badhaadhe ta’uun isaa barree, appilii biyya keessaaf carraa dursaa kennuun dhaamsa keenyaadha.
